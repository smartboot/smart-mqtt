# smart-mqtt 文档中心

欢迎来到 smart-mqtt 的官方文档中心！smart-mqtt 是 smartboot 开源组织推出的商业化 MQTT Broker 产品，专为拥有上万级设备连接量的企业级物联网场景设计。

## 为什么选择 smart-mqtt？

<CardGrid stagger>
    <Card title="超高性能" icon="rocket">
        单机支持百万级连接，消息吞吐量可达 790万/秒（QoS0），基于自研通信框架 smart-socket 打造。
    </Card>
    <Card title="纯国产血统" icon="badge-check">
        符合信创场景需求，底层采用自研通信框架，无外部依赖，代码可控。
    </Card>
    <Card title="插件化架构" icon="puzzle">
        基于事件总线的插件化设计，支持热插拔，可根据需求灵活扩展功能。
    </Card>
    <Card title="轻量级" icon="feather">
        v1.4.0 版本包体积不足 800KB，资源占用低，部署简单。
    </Card>
    <Card title="Java 生态" icon="seti:java">
        基于 Java 开发，天然适合 Java 技术栈企业，易于集成和二次开发。
    </Card>
    <Card title="企业级特性" icon="shield">
        支持集群部署、数据持久化、数据桥接、WebSocket、Dashboard 等企业级功能。
    </Card>
</CardGrid>
